Absensi — Sync via Gist

Index ini memuat loader Sync yang otomatis mengambil Gist ID & Token dari Google Docs publish-to-web.
Tips: di Google Docs, pilih File → Publish to the web, lalu ambil URL dan tambahkan ?output=txt.
Default app: absensi-settings.json. Case-sensitive.
Status: Belum mengambil kredensial
Buka Gist
• Kalau kamu sudah punya app absensi-nya, file ini bisa dijadikan “Sync Helper”. • Setelah Load, kamu bisa copy JSON ke app atau langsung Save untuk update storage di Gist. • Gunakan Ctrl/⌘+S di sini untuk cepat menyimpan ke Gist.
Keamanan: Token di dokumen publik berisiko. Minimal, jadikan Gist secret dan batasi masa aktif token.
